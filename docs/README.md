# 정책 문서

이 디렉토리에는 팀의 모든 정책 문서가 포함되어 있습니다.

## 📂 구조

- **policies/** - 현재 유효한 정책 문서
- **templates/** - 새 정책 작성을 위한 템플릿
- **archives/** - 과거 또는 폐기된 정책 (선택사항)

## 📝 정책 목록

### 현재 정책

- [예제 정책](./policies/example-policy.md) - 정책 작성 예시

## 🆕 새 정책 작성

1. 템플릿 복사: `cp templates/policy-template.md policies/새-정책.md`
2. 내용 작성
3. Pull Request 생성
4. 리뷰 및 승인
5. 병합 후 이 문서에 링크 추가

## 🔍 정책 검색

- GitHub 검색 기능 활용
- 키워드로 파일 내용 검색
- Git history로 변경 이력 추적

## 📊 정책 상태

- **초안**: 작성 중
- **검토중**: 리뷰 진행 중
- **승인됨**: 최종 승인 완료
- **폐기됨**: 더 이상 사용 안 함
